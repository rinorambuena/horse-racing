# 🏇 Horse Race Simulator

A lightweight, zero-dependency horse race simulator that runs entirely in the browser.

## Features

- 2 to 8 horses per race
- Customizable horse names and icons per horse
- Flip any horse horizontally by clicking it on the track
- 5 speed levels
- Randomized race outcomes — no two races are the same
- Podium with final standings

## Demo

[Play it here](https://rinorambuena.github.io/horse-racing/)

## Getting Started

### Option 1 — Open locally

Just download `index.html` and open it in any browser. No install, no server needed.

### Option 2 — Clone and run

```bash
git clone https://github.com/rinorambuena/horse-racing.git
cd horse-racing
open index.html
```

## How to use

1. Choose the number of horses
2. Set the race speed
3. Customize each horse's name and icon in the config panel
4. Click on any horse on the track to flip it horizontally
5. Press **Start** and watch the race

## Built with

Plain HTML, CSS, and JavaScript — no frameworks, no libraries.
